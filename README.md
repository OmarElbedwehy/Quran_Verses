`FEATURES`
- Random Ayah On Page Loaded Or Reloaded Or Click On Dice
- Translate From Arabic To English And Vice Versa
- Favourites List And You Can Add Or Remove Ayah From List
- Show Current Time
- Show Weather
- Show Current Date

`Tools Used`
- HTML
- CSS
- VANILA JAVASCRIPT
- JAVASCRIPT API
